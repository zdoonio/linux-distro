# Project XP-AI Linux Distribution Build Guide

## 🚀 Overview
Project XP-AI is a minimalist, lightweight Linux distribution based on Debian Stable (Bookworm). Its goal is to provide an extremely clean desktop experience styled after Windows XP's Luna theme while running on modern hardware. It includes core applications and a built-in AI Assistant skeleton for future expansion.

## 🛠️ Building the ISO Image
This system uses `live-build` to generate the final disk image.

**Prerequisites:**
Ensure you have installed all necessary tools: `live-build`, `fakeroot`, `quilt`.

**Steps:**
1. **Navigate to Project Directory:**
   ```bash
   cd ../linux-distro/project-xp-ai
   ```
2. **Configure Live Build (If necessary):**
   *This step assumes you ran `lb config` previously.*
3. **Build the Image:**
   Run the build command to generate the ISO file:
   ```bash
   sudo lb build
   ```

## 💾 Accessing the Output ISO
The resulting disk image will be placed in the `project-xp-ai/output/` directory (e.g., `project-xp-ai/output/live-*.iso`).

**To use the ISO:**
1. **Download/Locate:** Retrieve the `.iso` file from the `output` folder.
2. **Virtual Machine Setup:** Import this image into a virtual machine manager like VirtualBox or VMware.
3. **Installation:** Start the VM and follow the standard Debian installer prompts.

## ✨ Post-Installation Experience (Expected Features)
Once installed, you should observe:
*   **Visual Style:** XFCE Desktop Environment with Windows XP Luna theme/icons applied automatically.
*   **AI Assistant:** A dedicated AI assistant GUI window will launch automatically upon login, ready to receive input.
*   **Minimalism:** Only essential packages required for the core features are installed, keeping bloatware at bay.

## 📚 Customization Notes (For Developers)
*   **Adding Packages:** To include new applications, edit `config/package-lists/x86_64/standard.list.chroot` and list the package names.
*   **Theming/Icons:** Place custom theme files into the respective directories (`project-xp-ai/theme`, `project-xp-ai/icons`) to update the appearance.
